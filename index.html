My name is Jack, I'm a student, historian, musician, artist, and a couple more things that I'm probably not as committed to. This site is going to be for all my little interests and trivia, and I hope I can show someone something interesting :).

This site is free software! The license and source are available from the github repository, the link for which is to the left!
